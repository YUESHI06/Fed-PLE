# 模型效果整理

当前这一版实验主要围绕对比实验和消融实验展开。均匀假阴噪声条件下，RESCUER 的整体平均 F1 为 0.4560，在所有 baseline 中表现最好。非对称噪声条件下，CL 的整体平均 F1 为 0.4594，说明其对复杂噪声分布更敏感。完整模型 Ablation_Full 的整体平均 F1 为 0.4229，相比各消融变体保持了更稳定的综合表现。
