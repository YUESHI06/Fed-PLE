## 二、主实验核心结论

当前主实验已经比较清楚地说明了两件事：

1. `RESCUER-CEGT` 在均匀假阴噪声场景下整体效果最好，且稳定性最强。  
2. `Transformer` 和 `LCN` 这两个关键模块对最终性能提升具有明显贡献，其中 `Transformer` 的作用最强。

如果用一句话概括当前主实验结果，可以写成：

> 在真实 DAppSCAN 联邦漏洞检测任务上，所采用的 `RESCUER-CEGT` 方法在均匀假阴噪声条件下取得了最高的整体平均 F1，并在不同噪声强度下保持了更稳定的表现；同时，消融实验验证了 `Transformer` 表征模块和 `LCN` 标签修正模块对模型性能提升的关键作用。

## 三、对比实验结果

### 1. 均匀假阴噪声下的整体平均 F1

| 方法 | 0.0 | 0.1 | 0.2 | 0.3 | 平均 F1 |
| --- | ---: | ---: | ---: | ---: | ---: |
| RESCUER | 0.4439 | 0.4586 | 0.4663 | 0.4551 | **0.4560** |
| CL | 0.4328 | 0.4431 | 0.4410 | 0.3155 | 0.4081 |
| FedAvg | 0.4674 | 0.4882 | 0.3483 | 0.1250 | 0.3572 |
| ARFL | 0.4636 | 0.4449 | 0.3818 | 0.1176 | 0.3520 |
| FedCorr | 0.4518 | 0.3624 | 0.3774 | 0.1111 | 0.3257 |
| CLC | 0.2739 | 0.2739 | 0.2739 | 0.2739 | 0.2739 |

这组结果最重要的结论是：

- `RESCUER` 的整体平均 F1 最高，为 `0.4560`
- `CL` 排名第二，但在高噪声 `0.3` 时下降更明显
- `FedAvg / FedCorr / ARFL` 在低噪声下不差，但在高噪声下掉得很厉害
- `CLC` 整体最弱，基本没有形成有效竞争

因此，如果论文主线强调的是“联邦场景下对假阴噪声的整体鲁棒性”，当前主实验已经能支撑 `RESCUER-CEGT` 的优势。

### 2. 非对称噪声下的平均 F1

| 方法 | 平均 F1 |
| --- | ---: |
| CL | **0.4594** |
| RESCUER | 0.3917 |
| ARFL | 0.3518 |
| FedAvg | 0.3411 |
| CLC | 0.2739 |
| FedCorr | 0.1192 |

这组结果说明：

- 在非对称噪声条件下，`CL` 当前更强
- `RESCUER` 不是所有场景都第一，但仍保持较强竞争力
- 如果论文主结论放在“均匀假阴噪声下的联邦鲁棒检测”，当前结果是更有利的

## 四、消融实验结果

### 1. 各变体整体平均 F1

| 变体 | 0.0 | 0.1 | 0.2 | 0.3 | 平均 F1 |
| --- | ---: | ---: | ---: | ---: | ---: |
| Ablation_Full | 0.4029 | 0.4616 | 0.4611 | 0.3658 | **0.4229** |
| Ablation_woWarmup | 0.4513 | 0.4522 | 0.4536 | 0.3655 | 0.4307 |
| Ablation_woOrtho | 0.4988 | 0.4829 | 0.4108 | 0.1510 | 0.3859 |
| Ablation_woLCN | 0.4718 | 0.5018 | 0.3198 | 0.1071 | 0.3502 |
| Ablation_woTransformer | 0.2171 | 0.1635 | 0.1200 | 0.1208 | 0.1554 |

这一组里最值得强调的是：

- 去掉 `Transformer` 后，平均 F1 从 `0.4229` 直接降到 `0.1554`，说明图级语义表征是当前模型最核心的性能来源
- 去掉 `LCN` 后，平均 F1 降到 `0.3502`，说明标签修正模块对噪声学习是有效的
- `woOrtho` 和 `woWarmup` 也会影响结果，但影响程度没有 `Transformer` 那么强

### 2. Full 模型相对各模块的收益

| 变体 | Full - 当前变体 |
| --- | ---: |
| Ablation_woTransformer | 0.2675 |
| Ablation_woLCN | 0.0727 |
| Ablation_woOrtho | 0.0370 |
| Ablation_woWarmup | -0.0078 |

从这组结果看，当前最稳妥的写法是：

- `Transformer` 是决定性模块
- `LCN` 是明确有效的模块
- `Ortho` 有帮助
- `Warmup` 在当前这版平均值上没有拉开优势，但它对训练稳定过程仍然有作用，论文里更适合写成“训练策略组成部分”，不要把它写成最强贡献点

## 六、目前最能支撑的模型优点

基于当前已经完成的主实验，最能被结果直接支撑的优点有三点：

1. 在均匀假阴噪声联邦场景下，整体性能最好  
2. 在噪声升高时，整体结果更稳定  
3. `Transformer` 和 `LCN` 两个关键模块确实对性能提升有效

因此，当前主实验最适合支撑的表述不是“所有场景全面最优”，而是：

> 所提出方法在联邦智能合约漏洞检测的均匀假阴噪声场景下表现出更好的整体鲁棒性和更高的平均检测性能，同时关键模块的有效性也得到了消融实验验证。
